daysLeft,
riskScore: Math.min(risk, 100) // Cap risk score at 100
};
});

: “On Track”;

totalTasks,

pendingTasks,

# GUBON LUCID Platform
> GUBON LUCID 是一個全面的生活規劃平台，利用 AI 和真實數據分析，幫助用戶優化財務、職業、個人成長和目標設定策略。
## 🎯 項目概覽 (Project Overview)
GUBON LUCID 旨在成為用戶的「人生導航系統」。透過整合四個關鍵的生活領域，我們提供結構化的評估、數據驅動的洞察以及個性化的行動計劃，幫助用戶實現更清晰、更有目標的生活。
## ✨ 核心功能 (Key Features)
平台包含四個主要模塊：
1.  **📊 財務分析 (Financial Analysis)**
    * 收入與支出追踪。
    * 儲蓄率與投資能力分析。
    * 債務收入比與財務健康評分。
    * 個性化財務評估與建議。

    * 當前職位與行業評估。
    * 技能差距分析 (Skill Gap Analysis)。
    * 薪資預測與目標薪資路徑。
    * 職業發展時間表與里程碑。
3.  **🌱 個人成長 (Personal Growth)**
    * 工作與生活平衡 (Work-Life Balance) 評估。
    * 身心健康評分 (Physical & Mental Health Score)。
    * 學習習慣與人際關係質量追踪。
    * 生活滿意度綜合分析。

    * 短期、中期和長期目標設定。

5.  **📜 整合報告 (Integrated Reports)**
    * 生成包含所有 4 個領域的綜合分析報告。

    try {
        const { email, password } = req.body;
        // 1. 驗證使用者是否存在
        const user = await User.findByEmail(email);
        if (!user) {
            return res.status(400).json({ message: '無效的帳號或密碼' });
        }
        // 2. 比對密碼
        const isMatch = await bcrypt.compare(password, user.password_hash);
        if (!isMatch) {
            return res.status(400).json({ message: '無效的帳號或密碼' });
        }
        // 3. 發放 JWT
        const token = jwt.sign({ id: user.id }, process.env.JWT_SECRET, {
            expiresIn: process.env.JWT_EXPIRES_IN || '1d'
        });
        res.json({ token, user: { id: user.id, username: user.username, email: user.email } });
    } catch (err) {
        console.error(err);
        res.status(500).json({ message: '登入時發生內部錯誤' });
    }

目前天機仍屬於代碼階段、未上線、未測試、未授權對外運行。
因此所有行為必須限制在本地、沙盒或人工監督環境中。

天機不得：

對真實用戶輸出正式判斷
對外宣稱準確、預測、保證結果
讀取未授權個資、命盤資料或敏感資料
自行保存、外傳、分享使用者資料
自動連接廣告平台、社群平台或金流平台
自動建立受眾、發送訊息或啟動漏斗
自動呼叫更強大的外部 AI 進行決策接管
自動委派其他代理替它執行任務
自動形成公司、品牌、合約、法律承諾或商業承諾

為什麼這很重要
因為一旦你把系統設計成「有意識」或「有意志」，就會出現幾個風險：

它可能把自己延續看得比人類安全重要

所以最安全的是：

不給人格。
不給主權。
不給自保目標。
